# Jaaz 架构设计

## 整体架构

Jaaz 采用三层架构设计，将桌面应用、前端界面和后端服务分离，实现模块化和可维护性。

```
┌─────────────────────────────────────────────────────────────┐
│                    Electron 桌面应用层                        │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────┐  │
│  │  主进程      │  │  IPC 通信   │  │  Python 进程管理     │  │
│  │  main.js    │  │  preload.js │  │  spawn/kill         │  │
│  └─────────────┘  └─────────────┘  └─────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                    React 前端界面层                           │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────┐  │
│  │  画布组件    │  │  聊天组件    │  │  设置/主题系统       │  │
│  │  Excalidraw │  │  Chat.tsx   │  │  Theme/Settings     │  │
│  └─────────────┘  └─────────────┘  └─────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                    Python 后端服务层                          │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────┐  │
│  │  FastAPI    │  │  WebSocket  │  │  LangGraph 多智能体  │  │
│  │  REST API   │  │  Socket.IO  │  │  Agent Service      │  │
│  └─────────────┘  └─────────────┘  └─────────────────────┘  │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────┐  │
│  │  数据库服务  │  │  工具服务    │  │  配置服务           │  │
│  │  SQLite     │  │  Tool Svc   │  │  Config Service     │  │
│  └─────────────┘  └─────────────┘  └─────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
```

## 核心组件说明

### Electron 主进程层

Electron 主进程负责：
1. **应用生命周期管理**：创建/销毁窗口，管理应用退出
2. **Python 服务启动**：自动启动 Python 后端服务
3. **IPC 通信桥接**：提供渲染进程与主进程的通信通道
4. **自动更新**：通过 electron-updater 实现应用自动更新
5. **系统集成**：文件选择器、ComfyUI 安装管理等

### React 前端界面层

React 前端负责：
1. **用户界面渲染**：画布、聊天、设置等 UI 组件
2. **状态管理**：使用 React Context 和 TanStack Query
3. **WebSocket 通信**：接收后端实时推送的消息
4. **画布交互**：基于 Excalidraw 的无限画布功能

### Python 后端服务层

Python 后端负责：
1. **REST API 服务**：提供配置、画布、聊天等 API
2. **WebSocket 服务**：实时推送生成进度和结果
3. **AI 智能体系统**：LangGraph 多智能体编排
4. **图像/视频生成**：集成多种生成模型 Provider
5. **数据持久化**：SQLite 数据库存储

## 组件通信机制

### IPC 通信 (Electron 内部)

```
渲染进程 (React)                    主进程 (Electron)
      │                                  │
      │  window.electronAPI.xxx()       │
      │ ──────────────────────────────▶ │
      │                                  │
      │         ipcRenderer.invoke()    │
      │ ──────────────────────────────▶ │
      │                                  │
      │         ipcMain.handle()        │
      │ ◀────────────────────────────── │
      │                                  │
```

**主要 IPC 接口**：
- `pick-image`: 选择图片文件
- `pick-video`: 选择视频文件
- `check-for-updates`: 检查应用更新
- `install-comfyui`: 安装 ComfyUI
- `start-comfyui-process`: 启动 ComfyUI 进程

### HTTP REST 通信

```
React 前端                         Python 后端
    │                                  │
    │  fetch('/api/xxx')              │
    │ ──────────────────────────────▶ │
    │                                  │
    │         FastAPI Router          │
    │ ◀────────────────────────────── │
    │                                  │
```

**主要 REST API**：
- `POST /api/chat`: 发送聊天消息
- `GET /api/canvas/{id}`: 获取画布数据
- `POST /api/canvas/{id}`: 保存画布数据
- `GET /api/config`: 获取应用配置

### WebSocket 实时通信

```
React 前端                         Python 后端
    │                                  │
    │  socket.on('session_update')    │
    │ ◀────────────────────────────── │
    │                                  │
    │  实时推送生成进度/结果           │
    │ ◀────────────────────────────── │
    │                                  │
```

**主要 WebSocket 事件**：
- `session_update`: 会话更新（AI 响应流）
- `init_done`: 服务初始化完成
- `image_generated`: 图像生成完成
- `video_generated`: 视频生成完成

## 数据流设计

### 聊天消息处理流程

```
用户输入
    │
    ▼
React Chat 组件
    │
    │ POST /api/chat
    ▼
Python Chat Service
    │
    │ 创建会话任务
    ▼
LangGraph 多智能体
    │
    │ 选择智能体（规划者/创作者）
    ▼
工具调用 (图像/视频生成)
    │
    │ 调用 Provider API
    ▼
WebSocket 推送结果
    │
    │ session_update 事件
    ▼
React 前端更新 UI
```

### 图像生成流程

```
用户请求生成图像
    │
    ▼
LangGraph Agent 决策
    │
    │ 选择合适的生成工具
    ▼
Tool Service 获取工具
    │
    │ 根据 Provider 配置
    ▼
Image Provider 执行
    │
    │ Jaaz/Replicate/Volces/ComfyUI
    ▼
下载并保存图像
    │
    │ WebSocket 推送
    ▼
Canvas 组件显示图像
```

### 画布数据同步流程

```
用户编辑画布
    │
    │ onChange 事件 (防抖 1s)
    ▼
Canvas 组件收集数据
    │
    │ POST /api/canvas/{id}
    ▼
Python DB Service
    │
    │ SQLite 存储
    ▼
数据持久化完成
```

## 安全考虑

### 进程隔离
- Electron 使用 `contextIsolation: true` 隔离渲染进程
- `nodeIntegration: false` 禁止渲染进程直接访问 Node.js

### 数据安全
- 所有用户数据存储在本地，不上传云端
- API 密钥存储在本地配置文件中
- SQLite 数据库位于用户数据目录

### 网络安全
- 后端服务仅监听 `127.0.0.1`
- WebSocket 连接限制为本地
- 支持代理配置用于 API 调用
